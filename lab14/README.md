# Лабораторная работа 14: Веб-скрапинг контактов преподавателей РГПУ им. А. И. Герцена

Данная лабораторная работа реализует сбор контактной информации о преподавателях РГПУ им. А. И. Герцена с официального атласа вуза.

Сбор осуществляется с 1 по 54 страницу списка преподавателей, после чего для каждого преподавателя загружается его индивидуальный профиль для извлечения почты и телефона (при наличии). Собранные данные экспортируются в CSV-файл.

Задание реализовано двумя способами: на Python (BeautifulSoup) и на Node.js (Puppeteer).

---

## Способ 1: Python и BeautifulSoup

### Требования и установка зависимостей
Для работы скрипта требуются библиотеки `requests` и `beautifulsoup4`. Их можно установить командой:
```bash
pip3 install beautifulsoup4 requests
```

### Запуск
Для запуска скрипта выполните команду:
```bash
python3 scraper_bs4.py
```

### Принцип работы
1. Скрипт последовательно проходит по страницам списка с 1 по 54, собирая ФИО преподавателей и ссылки на их профили.
2. Для ускорения обхода 2160+ профилей используется многопоточность (`ThreadPoolExecutor` на 15 потоков).
3. Данные сохраняются в файл `teachers_bs4.csv` в корне папки `lab14`.

---

## Способ 2: Node.js и Puppeteer

### Требования и установка зависимостей
Необходимо установить библиотеку `puppeteer`. Для этого запустите команду:
```bash
npm install
```

### Запуск
Запустите скрапер командой:
```bash
npm start
```
или:
```bash
node scraper_puppeteer.js
```

### Принцип работы
1. Запускается безголовый (headless) браузер Chromium.
2. Страницы списка (1-54) загружаются параллельно пачками по 5 страниц. Из них извлекаются ссылки на профили.
3. Профили преподавателей обходятся пачками по 15 вкладок параллельно.
4. Для экономии трафика и ускорения загрузки перехватываются запросы и отменяется загрузка картинок, стилей и шрифтов.
5. Результаты сохраняются в файл `teachers_puppeteer.csv` в корне папки `lab14`.
