Ход выполнения лабораторной работы

Шаг 1
Установка Yandex SourceCraft Code Assistant
Скачали VSIX плагина с официального сайта SourceCraft и установили его в VSCode с помощью CLI-команды code --install-extension. В боковой панели отобразилась инструкция по авторизации.
Шаг 2
Авторизация в Code Assistant
Провели сопряжение VSCode с личным кабинетом Yandex Cloud для полноценной работы AI-помощника генерации кода.
Шаг 3
Установка плагина Yandex Cloud Toolkit
Изучили репозиторий Yandex Cloud MCP серверов и сконфигурировали сервер Toolkit для интеграции со средой разработки, добавив настройки в mcp_config.json.
Шаг 4
Настройка конфигурации MCP-сервера
Определили параметры командного вызова npx для запуска серверов Toolkit и Containers по протоколу STDIO.
Шаг 5
Подключение Yandex Cloud Containers
Настроили MCP-сервер Containers для автоматизации операций с Docker-реестрами и развертыванием бессерверных контейнеров.
Шаг 6
Деплой в Yandex Serverless Containers
Для автоматического развертывания веб-приложения использовали Yandex Cloud CLI (yc) и Docker-образ, запушив его в реестр flask-app-registry.
yc serverless container revision deploy \
  --container-name flask-app-container \
  --image cr.yandex/crpq8o.../flask-app:latest \
  --cores 1 --memory 128MB --concurrency 16
